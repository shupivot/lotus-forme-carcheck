② 开发者模式 / ADB 探路（车机系统里手动操作）

  
先看这条：浏览器无法开启也不能确认开发者模式和 ADB——那是系统层设置，在浏览器沙箱之外。下面只是吉利 / ECARX 系车机的候选路径，ForMe 的当前版本未证实。只做只读检查；若入口不存在就停止，不刷机、不输来源不明密码、不安装工程包。


  

    

      
A
版本号连点（最常见）

        
车机 设置 → 系统 / 关于本机 → 连续点击「版本号 / 系统版本」5~7 次 → 弹出「已进入开发者模式」→ 返回找到「开发者选项」→ 打开「USB 调试 / 无线调试(ADB)」。

      
    

    

      
B
时钟连点进工程菜单

        
部分 ECARX 车机：连点屏幕右上角时钟约 10 次进工程菜单 → 找「ADB / USB 模式切换」→ 选「Host mode（主机模式）」。

      
    

    

      
C
无线 ADB（车机最常用，无 USB 口时走这个）

        
开发者选项里打开「无线调试」后：电脑与车机连同一个手机热点 → 电脑执行 adb connect 车机IP:5555 → 车机弹窗点「允许」。车机 IP 在 设置→网络 里看。

      
    

    

      
D
侧载 App（拿到 ADB 之后）

        
连上后 adb install 应用.apk 装第三方应用；或用「甲壳虫」类工具远程装。注意：HYPER OS 的「安全管家→应用管理」可能拦截未知来源，需先在里面放行。
